02

Colour

Saudi green is the accent, used sparingly. Gold is the luxury moment. Neutrals carry 90% of the surface area. Status colours are tuned for both light and dark themes (the dark theme lifts accent + status to remain legible on dark backgrounds).

04

Buttons

One button family with four variants × three sizes. Hover lifts shadow, active drops 1px and 99% scale. Focus ring uses the accent's 35%-alpha glow.

07

Right-to-left rendering

Same components, mirrored via dir="rtl". The font token automatically swaps to IBM Plex Sans Arabic. Logical CSS properties (margin-inline-start) handle the mirroring for everything not absolutely positioned. Numbers use bidi isolate so they read correctly within Arabic prose.

08

Elevation & motion

Six shadow tokens, each two-layered. Default motion easing is Linear-style cubic-bezier(0.16, 1, 0.30, 1); default duration 220ms. Reduced-motion flattens both.
